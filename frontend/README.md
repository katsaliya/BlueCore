
  # Voice-Activated Maritime Consultant App

  This is a code bundle for Voice-Activated Maritime Consultant App. The original project is available at https://www.figma.com/design/1qwG5VAPlDlRwIU80OrqnG/Voice-Activated-Maritime-Consultant-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Copy `.env.example` to `.env` if the backend is not running at `http://127.0.0.1:8080`.

  Run `npm run dev` to start the development server.
